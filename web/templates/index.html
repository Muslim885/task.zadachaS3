Welcome to TaskS3 - a complete image management system with S3 storage.

Available Endpoints:

  • /health - Health check endpoint
  • /api/images - List all images in S3
  • POST /api/upload - Upload new image (use form-data with "image" field)
  • POST /api/process - Process local images
  • POST /api/move - Move images in S3
